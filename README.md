Student Grade Calculator (Python)
Description
This is a simple Python program that calculates the grade of a student based on the marks entered by the user. The program accepts marks between 0 and 100 and returns the corresponding grade.
Grade Criteria
* 75 – 100 : A
* 65 – 74 : B
* 55 – 64 : C
* 35 – 54 : S
* 0 – 34 : F
How It Works

1. The user enters the marks (0–100).
2. The program checks the mark range.
3. It assigns the correct grade.
4. The grade is displayed to the user.

Technologies Used
Python

Example

Input:
Enter your marks (0-100): 78

Output:
Your grade is: A
